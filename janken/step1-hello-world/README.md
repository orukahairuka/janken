# Step 1: Hello World

## 学習ポイント

### 1. Text ビュー
`Text` は文字を表示するための最も基本的なビューです。
```swift
Text("じゃんけんアプリ")
```

### 2. VStack 
`VStack` は複数のビューを縦に並べるためのコンテナです。
- V は Vertical（垂直）の略
- 今回は Text が1つだけですが、後で複数のビューを追加します

### 3. 修飾子（Modifier）
ビューの見た目を変更するメソッドです。
- `.font(.largeTitle)` - 文字を大きくする
- `.foregroundColor(.blue)` - 文字の色を青にする

### ポイント
- 修飾子は連鎖できます（チェーンできます）
- 順番によって結果が変わることがあります
- Xcodeの自動補完を活用しましょう
